# 如何读取文件权限？
本文介绍文件的权限设置。vimrc列表中有趣的权限包括：
```bash
rw-r--r–
```
这个字符串实际上是三组不同权限的表达：
```bash
rw-
r--
r--
```
1. **第一组权限适用于文件的所有者。第二组权限适用于拥有该文件的用户组。第三组权限通常称为“其他”。所有 Linux 文件都属于所有者和组。**

2. **当权限和用户用字母表示时，这称为符号模式。对于用户，u代表用户所有者、g组所有者和o其他人,a代表所有。对于权限，r代表读取、w写入和x执行。**

# 什么是八进制值？
当 Linux 文件权限用数字表示时，这称为数字模式。在数字模式下，三位数值表示特定文件权限（例如 744）。这些称为八进制值。第一位表示所有者权限，第二位表示组权限，第三位表示其他用户权限。每个权限都分配有一个数值：

- r（读）：4
- w（写入）：2
- x（执行）：1


# 如何修改Linux文件权限？
您可以使用命令chmod（代表“更改模式”）修改文件和目录权限。要在数字模式下更改文件权限，请chmod在文件名旁边输入所需的八进制值（例如 744）。要在符号模式下更改文件权限，请在文件名旁边输入用户类别和要授予他们的权限。例如：
```bash
$ chmod ug+rwx example.txt
$ chmod o+r example2.txt
```
## sample：
```bash
$ chmod a+x test.bin
```
>a是指所有的用户组，包括root用户组，文件拥有者的用户组，还有其他用户组。
>+x是指添加执行权限。
>+x是执行权限，+r是阅读权限，+w是写入权限
